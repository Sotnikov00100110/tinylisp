# TinyLisp in C

Мінімалістичний інтерпретатор мови Lisp, написаний на C. Проєкт створений для вивчення основ функціонального програмування та роботи з низькорівневою пам'яттю.

## Особливості
- Підтримка списків (Cons Cells).
- Базова арифметика (+, -, *).
- Робота зі змінними через `def`.
- Глобальне середовище виконання (Global Environment).

## Як скомпілювати
Використовуй Make:
```bash
make
./lisp "(+ 1 2 3)"

## Для використання списків і виразів по типу def, car, cdr можете використовувати наступне 
- (def x 10) (def y 15) (+ x y) 
    Result: 25
- (car (12 2 3)) 
    Result: 12 
- (cdr (12 2 3))
    Result: 2 3

## Що потрібно реалізувати 
Потрібно додати наступний функціонал:
- підтримку hash map для повноцінного і швидкого парсингу
- додати власний алокатор (по типу bump allocator або mark and sweep алгоритм)
- відформатувати код і розбити його на структури, окремо винести константи в types.h
